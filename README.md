# simple-static-startpage
A stupidly simple, static, startpage for selfhosted services.

## Usage
Edit config.json with your services:
`vim config.json` 

Start the docker container:
`docker-compose up -d`

Browse to your startpage:
`open http://<your-ip>:8080`
